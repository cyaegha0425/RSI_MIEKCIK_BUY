================================================================================
                    咩咩蹄到好船来 V3.0.2 使用指南
================================================================================

欢迎使用咩咩蹄到好船来！这是一款 RSI 星际公民商店的自动抢购工具。
本指南面向零基础用户，从打开程序到完成抢购，一步步带你走完。

================================================================================
                         第一部分：准备工作
================================================================================

在运行程序之前，你需要确认以下事项：

1. 你的电脑是 Windows 10 或 Windows 11 系统
2. 已安装 Microsoft Edge 浏览器
3. 打开 Edge 浏览器，访问 robertsspaceindustries.com 并登录你的 RSI 账号
   （登录状态很重要！程序需要用到你的登录信息）
4. 确认你的 RSI 账户有足够的信用点余额
5. 确认你的 RSI 账户已设置默认账单地址

================================================================================
                         第二部分：运行程序
================================================================================

1. 解压下载的压缩包到任意文件夹
2. 双击运行 RSI_MIEKCIK_BUY.exe
3. 程序启动后会显示配置窗口

如果 Windows 提示"已保护你的电脑"，点击"更多信息"→"仍要运行"。

================================================================================
                         第三部分：填写配置
================================================================================

程序打开后，你会看到以下配置项：

【输入模式】

  刷新拦截模式（推荐）：
    - 填写搜索关键词，比如你想买 Kraken 就填 Kraken
    - 程序在开抢瞬间刷新页面，自动识别并加购
    - 如果搜索结果有多种商品，可以用"排除关键词"排除不想要的
    - 例：搜 Kraken 时排除 Privateer，就只匹配非Privateer版本
    - 不需要提前知道 SKU ID
    - 价格为可选项：不填则程序自动获取，填入后速度更快
    - 如果手动填入价格，请确保价格准确，否则付款会失败

  SKU ID 直购模式：
    - 直接输入商品的 SKU ID 和价格
    - 价格必须准确填写，否则付款会失败
    - 适合已经知道 SKU ID 的情况
    - 可以点击"收藏夹"按钮从已保存的商品中选择

【抢购时间】

  填写商品开售的时间，格式：年-月-日 时:分:秒
  例：2026-05-15 00:00:00

  程序会自动校准时间差，你只需要填写 RSI 官方公布的开售时间即可。

【伏击模式】（可选）

  如果你要抢的商品已经加入了购物车，可以勾选伏击模式：
  - 程序会提前完成大部分结账步骤
  - 开抢瞬间只需要发送最后一条确认请求
  - 速度比正常模式更快

  注意：使用伏击模式前，请确保购物车里只有目标商品！

================================================================================
                         第四部分：抢购流程
================================================================================

填写完配置后，点击"开始抢购"按钮，程序会自动执行：

1. 启动浏览器
   - 程序会自动打开 Edge 浏览器并连接你的 RSI 登录状态

2. 打开商品页面
   - 程序会预先打开 RSI 商店页面进行预热

3. 提醒清空购物车
   - 弹窗提醒你确认购物车已清空
   - 请在浏览器中手动清空购物车，然后点击确认

4. 时间校准
   - 程序自动测量你电脑和 RSI 服务器之间的时间差
   - 校准完成后会显示偏移值

5. 等待开抢
   - 倒计时期间你不需要做任何事情

6. 开抢瞬间
   - 拦截模式：刷新页面 → 自动识别商品 → 加购 → 付款
   - SKU 模式：提前 10 秒开始尝试加购 → 加购成功 → 付款

7. 显示结果
   - 成功：显示订单号
   - 失败：显示失败原因

================================================================================
                         第五部分：SKU 收藏夹
================================================================================

程序会自动记录拦截到的商品信息，保存到收藏夹中。
首次使用时，程序会自动加载预置的商品列表。

【查看收藏夹】
  点击配置界面 SKU ID 行旁边的"收藏夹"按钮

【收藏夹功能】
  - 搜索：输入关键词筛选商品
  - 选择：点击"选择"按钮，自动填入 SKU ID 和价格
  - 删除：点击"×"按钮移除不需要的条目
  - 手动添加：在底部输入名称、SKU ID 和价格，点击"+添加"
  - 鼠标滚轮翻页：在收藏夹区域内滚动鼠标滚轮即可

【自动保存】
  每次使用拦截模式成功拦截到商品后，所有识别到的商品都会自动保存。
  下次可以直接从收藏夹选择，不需要重新查找 SKU ID。

================================================================================
                         第六部分：日志窗口
================================================================================

抢购过程中，你可以点击进度界面左下角的"📋 日志"按钮打开日志窗口。

日志窗口会实时显示程序运行过程中的详细信息，包括：
  - API 请求和响应
  - 时间校准的采样数据
  - 加购和付款的详细步骤
  - 错误信息

如果抢购出现问题，日志窗口是排查问题的第一选择。

================================================================================
                         第七部分：常见问题
================================================================================

问：程序提示"登录失败"怎么办？
答：用 Edge 浏览器手动访问 RSI 网站检查是否还处于登录状态。
    如果已退出，重新登录后再运行程序。

问：程序提示"SKU 直购模式必须手动填写价格"怎么办？
答：SKU 直购模式必须准确填写商品价格（美元），价格不正确会导致付款失败。
    你可以在 RSI 官网商品页面查看价格，或者从收藏夹选择（会自动填入价格）。

问：时间校准失败怎么办？
答：检查网络连接是否正常。如果持续失败，可以在高级设置中使用手动偏移。

问：抢到了但没付款成功？
答：最常见的原因是信用点余额不足或默认地址未设置。
    请在 RSI 网站确认账户状态。

问：程序界面卡住不动了？
答：程序在后台执行抢购，界面暂时无响应是正常的。等待结果即可。
    如需强制退出，按 Ctrl+C 或直接关闭窗口。

问：如何获取 SKU ID？
答：最简单的方法是先用拦截模式抢一次，程序会自动保存所有发现的 SKU ID。
    也可以在收藏夹弹窗底部手动添加。

问：信用点金额和商品价格不一致会怎样？
答：付款会失败。信用点的金额必须和商品价格完全一致。
    所以 SKU 模式下务必确认价格填写正确。
